# PromptLayer Icons

This folder contains the extension icons in various sizes required by Chrome Web Store.

## Required Sizes

- `icon-16.png` - 16x16 - Extension toolbar icon
- `icon-48.png` - 48x48 - Extension management page
- `icon-128.png` - 128x128 - Chrome Web Store listing

## Temporary Placeholders

The current icons are SVG placeholders. For production, replace these with proper PNG icons.

## Creating Icons

You can use tools like:
- Figma
- Adobe Illustrator  
- Inkscape (free)
- Canva

### Design Guidelines

- Use the PromptLayer brand color: `#10a37f`
- Keep the design simple and recognizable at small sizes
- Use a transparent background
- Export as PNG with proper dimensions
